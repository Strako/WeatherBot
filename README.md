# WeatherBot
A weather bot for Discord

Readme ! 
--------------------------------
-Requirements

	Python 3.10.6 Lastest
--------------------------------
	-Dependencies:
	
		Discord.py
			Python3.10 -m pip install -U discord.py
			--------
			verify installation 
			pip3 show discrod.py 
				or
			pip show discord.py		

		Certifi:
			pip3 install certifi
			--------
			verify installation 
			pip3 show certifi 
				or
			pip show certifi
